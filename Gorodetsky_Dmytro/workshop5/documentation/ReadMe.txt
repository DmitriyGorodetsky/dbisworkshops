Городецький Дмитро КМ-72 Project(Workshop5)
Порядок запуску:
1. models.py
2. packages_and_populate.sql
3. main.py


Зміст проекту - веб-застосунок для можливості пошуку книг за умови реєстрації користувача
Реалізовано:
реєстрація користувача;
авторизація зареєстрованого користувача;
можливість пошуку книг;
можливість додати книгу до кошику замовлень;
переглянути  статистику бази даних.
